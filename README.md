# Shell_Automation_Scripts

## Overview
This repository contains **8 Bash scripts** developed as part of **Weekend Shell Scripts Challenge**.  
The scripts automate common tasks such as backups, disk monitoring, deployment simulation, process checking, log analysis, file cleanup, API data fetching, and cron simulation.  

All scripts were developed and tested on **MSYS/Zsh**.
